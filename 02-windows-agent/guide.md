# Phase 2 — Intégration d’un poste Windows avec Sysmon

## Objectif

L’objectif de cette phase est d’intégrer une machine Windows 10 au serveur Wazuh afin de remonter des événements de sécurité.

Un outil complémentaire, Sysmon, est utilisé pour enrichir la visibilité sur les activités système.

Cette phase permet de :

* superviser un poste Windows
* collecter des événements avancés
* améliorer la détection des comportements suspects
* préparer la simulation d’attaques

La machine Windows deviendra la cible principale des scénarios d’attaque dans les phases suivantes.
